¿Que es html?

  • HTML significa Hyper Text Markup Language
  • HTML describe la estructura de una página web
  • HTML consiste en una serie de elementos
  • Los elementos HTML le dicen al navegador cómo mostrar el contenido
  • Los elementos HTML están representados por etiquetas
  • HTML etiquetas de la etiqueta de piezas de contenido como "mesa" "la partida", "párrafo", y así sucesivamente
  • Los navegadores no muestran las etiquetas HTML, pero las usan para representar el contenido de la página

Imagenes

En HTML las Imagenes estan definidas con el tag < img> y esta compuesta por los siguientes atributos

  • src = aqui ponemos la ruta relativa de la imagen que queremos mostrar
  • alt = aqui ponemos el texto que queremos que se muestre en caso de que no se pueda cargar la imagen
  • width = aqui definimos la anchura de la imagen en pixeles
  • height = aqui definimos la altura de la imagen en pixeles

Atributos HTML

Definicion

  • Todos los elementos HTML pueden tener atributos
  • Atributos proporcionan información adicional acerca de un elemento
  • Atributos siempre se especifican en la etiqueta de inicio
  • Atributos general vienen en pares nombre / valor como: name = "valor"

Ejemplos

  • href = definido por el tag < a> sirve para agregar links de otras paginas
  • src = definido por el tag < img> sirve para agregar fotos
  • width y height = definido por el tag < wight> < height> sirve para controlar el tamaño en pixeles de las imagenes
  • alt = definido por el tag < img> sirve para señalar el texto a mostrar cuando no se puede mostrar la imagen
  • style = es el atributo usado para especificar el estilo de un elemento como el color, la fuente, el tamaño, etc
  • lang = indica el idioma de la pagina para que Google pueda ofrecer la traduccion. Esta compuesta por dos partes, la primera especifica el lenguaje, la segunda especifica el dialecto. En España es "es-ES"
  • title = sirve para indicar el texto a mostrar cuando ponemos el cursor encima de algo

Tablas

Definicion

Con el tag < table> podemos generar tablas html

Parametros con ejemplos

Crear una tabla

  • Se utiliza el tag < tr> para definir las filas

  • Con el tag < th> definimos el encabezado de la tabla, por defecto este viene en negrita y centrado

  • La tablas se rellena con datos que van entre los tags < td>

  • Si no se especifica un borde, por defecto la tabla vendra sin estos, para definir un borde usamos la propiedad "border" dentro de nuestro CSS

  • Si quieres que los bordes esten colapsados dentro de nuestro CSS tenemos que añadir la propiedad "border-collapse"

  • Con la propiedad CSS "padding" podemos añadir padding a las celdas para agrandar el tamaño de estas

  • Por defecto el encabezado de la tabla esta centrado pero con las propiedad CSS "text-align" podemos cambiar eso

  • Es parecido al padding pero esta vez amentamos la distancia entre los bordes de las celdas, usamos "border-spacing"

  • Para hacer que una celda se espanda mas de una columna, usamos el atributo "colspan", este atributo tiene una variable llamada "rowspan" con el cual podemos hacer que las celdas utilizen mas de una fila

  • Para añadir un titular a la tabla usamos el tag < caption>

  • En las tablas tambien se puede usar el atributo "id" para poder cambiarle los colores entre otras opciones en el CSS

Bloques y elementos en linea

Definicion

Cada elemento HTML tiene un valor de visualizacion por defecto dependiendo de que tipo de elemento es

Ejemplos de elementos de bloque

  • address: define la informacion de contacto del autor/propietario de un documento o articulo
  • article: define contenido independiente y auto-contenido
  • aside: define cierto contenido a parte del contenid donde se encuentra
  • blockquote: defne una cita de otra fuente
  • canvas: sirve para dibujar graficos
  • dd: sirve para describir un termino/nombre en una lista de descripcion, la lista se define con el tag < dl> y cada termino se define con < dt>
  • div: define una seccion o division del documento HTML
  • fieldset: sirve para crear campos para rellenar
  • figure: define contenido como ilustraciones, diagramas, fotos, etc...
  • figcaption: define el pie de foto para un elemento < figure>
  • footer: define el pie de pagina del documento
  • form: es un tag utilizado para definir una zona en la que se puede introducir texto
  • h1-h6: definen el tamaño de un encabezado
  • header: es un elemto que representa un contendor para introducir contenido o un set de links de navegacion
  • hr: separa una parte del documento de otra, dibujando una linea a lo ancho del documento
  • li: define un item dentro de una lista
  • main: define el contenido principal de un documento
  • nav: define un set de enlaces de navegacion, NO TODOS LOS ENLACES DEL DOCUMENTO TIENEN QUE IR DEFINIDOS EN ESTE TAG, SOLO LOS ENLACES PRINCIPALES
  • noscript: define el contido para los usuarios que tienen los scripts desactivados en el navegador o que su navegador no los soporta
  • ol: define una lista ordenada
  • p: define un parrafo
  • pre: define texto pre-formateado
  • section: define secciones en el documento como capitulos, encabezados, pies de pagina o otra seccion del documento
  • thead, tbody, tfoot: no afectan a la estructura de la tabla, sin embargo los puedes usar para definir estilos CSS
  • ul: define una lista desordenada
  • video: especifica video, con el tag type podemos definir el formato del video que pueden ser MP4, WebM, Ogg

Ejemplos de elementos en linea

  • a: define un hyperenlace que es usado para linkear una pagina con otra
  • abbr: define una abreviacion o acronimo
  • acronym: es igual que el abbr salvo que no no tiene soporte en HTML5
  • b: define texto en negrita
  • bdo: define la direccion en la que se escribe un texto
  • big: define texto mas grande pero no tiene soporte en HTML5
  • br: define un salto de linea
  • button: define un boton pulsable
  • cite: define el titulo de un trabajo(libro, cancion, pelicula, programa de television, pintura, escultura)
  • code: define un trozo de codigo de ordenador
  • dfn: defina la instancia de definicion de un termino en HTML
  • em: define texto enfatizado
  • i: define texto en italica
  • img: define una imagen en una pagina HTML
  • input: define un campo de texto donde el usuario puede introducir datos
  • kbd: defina la entrada de teclado(estilo de texto)
  • label: define una etiqueta para:
    • <"button">
    • < input>
    • < meter>
    • < output>
    • < progress>
    • < select>
    • < textarea>

    No renderiza nada especial para el usuario. Lo que si que hace es mejorar la el uso para usuarios con ratones

  • map: define un mapa de imagenes
  • object: define un objeto insertado en el documento HTML, usa este elemento para insertar multimedia en tus paginas web
  • output: define le resultado de un calculo(una realizado por un script por ejemplo>
  • q: define una breve cita
  • samp: define salida de texto del un programa del ordenador
  • script: define un script en el lado del cliente
  • select: elemento que es usado para crear una lista
  • small: define texto pequeño
  • span: se utiliza para agrupar elemetos en linea dentro de un documento
  • strong: define texto importante
  • sub: define subtexto
  • sup: define supertexto
  • textarea: define una area de texto en la que puedes introducir un numero ilimitado de caracteres
  • time: define una fecha/hora legible, tambien se puede utilizar para codificar las fechas y los horarios de forma legible por la maquina
  • tt: es como el < kbd> pero no tiene soporte en HTML 5
  • var: define una variable
